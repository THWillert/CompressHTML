Plugins selber machen

Alle Algorithmen müssen sich im Verzeichnis Plugins befinden.
Jedes Plugin besteht aus zwei Dateien:
  • Dem eigentlichen Plugins (name.rexx)
  • Einer Beschreibung (name.txt), wird diese weggelassen erzeugt das Programm eine Standardinformation, aus einer Anfrage an das Plugin und dessen Versions-String.
Ansonsten einfach die Datei Vorlage_Modul.rexx ansehen.

Wer mehr Informationen benötigt, soll sich doch bitte mir in Verbindung setzen.
Ich werde dann auch diese Seite, dahingehend erweitern.
