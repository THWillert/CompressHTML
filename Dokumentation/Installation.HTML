Installation

Das komplette CompressHTML Verzeichnis kann einfach an die gewünschte Stelle kopiert werden.

Vor dem ersten Start von CompressHTML, das Programm rx-patcher im gleichnahmigen Verzeichnis starten.
(rx-patcher erzeugt in REXXC:, ein Programm das sich RXNIL nennt. Dieses öffnet im Gegensatz zu RX kein CON-Fenster!)
